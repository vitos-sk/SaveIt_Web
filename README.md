# SaveIt - Telegram Mini App

Telegram Mini App для просмотра данных пользователя из Firebase Realtime Database.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Функционал

- Подключение к Firebase Realtime Database
- Получение данных текущего пользователя Telegram
- Отображение структуры базы данных
- Черный фон UI в стиле Telegram Mini App
